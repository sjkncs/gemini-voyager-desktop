## Gemini Voyager Desktop Edition v1.2.4

Electron desktop wrapper for [Gemini Voyager](https://github.com/Nagi-ovo/gemini-voyager) — the all-in-one Google Gemini enhancement suite.

### Highlights

- **Full feature parity** with the browser extension — no Chrome/Firefox required
- **Chrome API polyfills** — `storage.sync`, `tabs`, `permissions`, `identity`, `scripting`
- **Custom website injection** via `voyager-ipc://` protocol bridge
- **Settings export/import** via native file dialogs (replaces Chrome Sync)
- **MV3 → MV2 auto-patching** for Electron compatibility
- **Cross-platform** — Windows, macOS, Linux

### All Supported Features

- Timeline Navigation, Folder Organization, Prompt Vault
- Chat Export (JSON/MD/PDF), Mermaid Rendering, Markdown Fix
- NanoBanana watermark removal, Deep Research Export
- Quote Reply, Batch Delete, Tab Title Sync
- Hide Recents & Gems, Input Collapse, Default Model
- CoBridge Context Sync (IDE integration)
