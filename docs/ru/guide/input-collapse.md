# Сворачивание ввода

Сворачивайте поле ввода, когда оно пустое, чтобы получить больше пространства для чтения. Нажмите на свернутую панель, чтобы развернуть и начать печатать.

</div>

## Как использовать

1. Когда поле ввода пустое и теряет фокус, оно автоматически сворачивается в компактную кнопку-таблетку
2. Нажмите на кнопку-таблетку, чтобы развернуть поле ввода и начать печатать
3. Вы можете включить или отключить эту функцию в панели настроек (отключено по умолчанию)
