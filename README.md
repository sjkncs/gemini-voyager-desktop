This is an **Electron desktop wrapper** for [Gemini Voyager](https://github.com/Nagi-ovo/gemini-voyager), the all-in-one enhancement suite for Google Gemini. It packages the browser extension into a standalone desktop application with full feature parity â€” plus additional desktop-specific enhancements.

### Why a Desktop Version?

- **No browser extension installation** â€” just launch and use
- **Dedicated window** â€” separate from your browser, always accessible
- **Native OS integration** â€” system tray, keyboard shortcuts, file dialogs
- **Settings export/import** â€” backup and restore via native file picker (replaces Chrome Sync)
